# Adaptive Experimentation

**Adaptive, safety-first traffic allocation for real-world experimentation.**

This documentation expands on the concepts, design principles, and examples
provided in the main repository.

---

## What this project is about

Adaptive experimentation helps systems learn which variants perform better
*over time*, while avoiding aggressive or unsafe behavior.

This library focuses on:
- conservative defaults
- explainable decisions
- operational realism
